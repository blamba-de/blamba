What is this?
A ringtone provider, similar to popular commercial providers from the early 2000s.

Is this real?
Yes! If you have a supported mobile phone (mostly Nokia at this point), you can send all the content available directly to your phone.

Where does this work?
Blåmba needs some mechanism to send raw binary data to participants. We can do this without creating any costs in the private GSM networks at hacker conferences (like CCC events or EMF).

Does this cost money?
No! No sneaky subscriptions, either :)

Which phones are supported?
Nokia 6310i, Nokia 6220 and Nokia 6233 were used for development.
Many other Nokia phones will work fine as well.
Other vendors (Sony Ericsson, etc.) aren't implemented due to a lack of devices (and knowledge about them). If you have such a phone and know how to encode data for it, please contact us!

My phone needs a WAP/WSP gateway / IP address settings!
Blåmba operates a public WAP gateway, which can be used via GPRS from the phone.
It uses an IP / ASN-based allowlist to prevent abuse. Most german phone providers and the CCC event network are allowed.
The IPv4 address is: 157.90.145.189 (no username/password).
If you're using a different carrier/provider, please write us a message with your public IPv4 address or the ASN of your carrier.

How does this work?
Operator logos und RTTTL ringtones will be sent via SMS.
MIDI files, J2ME applications and games will be sent via WAP-Push (basically a special link via SMS).
If you're interested in the nitty-gritty details, the source code is linked in the menu.

I have a cool logo/ringtone/game/application.
Awesome! Please send a pull request to the blamba-content-ingest repository.

I found a security vulnurability.
Please send an e-mail to blamba@tbspace.de.

My phone doesn't support WAP push (e.g. iPhone). What can I do?
Prefix the catalog number with "plain" ("plain 12345").
Blåmba! will then send a regular text SMS with an HTTP-URL.

What are operator logos? What are group logos?
Operator logos are little logos, which are assigned to a specific mobile network.
When a phone connects to a specific network, it will display the logo for this carrier on the home screen.
The carrier/network will be detected using the globally unique MCC/MNC IDs.
The CCC Event network usually uses 262/42. Deutsche Telekom uses 262/01.

Group logos use the same graphics format and encoding, but can be assigned to contacts and contact groups. They will then be displayed when such a contact or contact group calls.
